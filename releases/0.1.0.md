### JEAF Timetable Service Model

#### [#577](https://github.com/anaptecs/jeaf-generator/issues/577) SoftLinks for derived properties ![Backward Compatible](https://www.jeaf-generator.io/icons/backward-compatible-24.png) ![Strict Forward Compatible](https://www.jeaf-generator.io/icons/strict-forward-compatible-24.png)
Now it is also possible to use SoftLinks for derived properties inside the UML model.

<br>

#### [#676](https://github.com/anaptecs/jeaf-generator/issues/676) Improve interoperability of generated code with other classes also using Jackson ![Backward Compatible](https://www.jeaf-generator.io/icons/backward-compatible-24.png) ![Strict Forward Compatible](https://www.jeaf-generator.io/icons/strict-forward-compatible-24.png)
New behavior can be activated be setting configuration parameter `generateJsonSetterAnnotationOnBuilderOperations` to `true` in your Maven Plugin Configuration for JEAF Generator.
